# Colgate Smile App

The **Colgate Smile App** is a Flask-based web application that analyzes a user's smile and applies a whitening effect to their teeth.  
It uses **OpenCV**, **Mediapipe**, and **Pillow** to detect facial landmarks, isolate the teeth region, and apply whitening while keeping the smile natural.
